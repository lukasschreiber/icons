# @lukasschreiber/icons

React SVG icon library.

## Installation

``` bash
npm install @lukasschreiber/icons
```

## Usage

``` tsx
import { Search } from "@lukasschreiber/icons";

export default function Example() {
  return <Search size={24} />;
}
```

## Features

-   9,000+ SVG icons
-   Tree-shakable
-   React 18+
-   TypeScript support
